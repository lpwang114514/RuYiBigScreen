# RuyiBigScreen

如意数据大屏是一个面向零基础学习者的公开数据可视化大屏项目，目标是帮助大家从 0 到 1，自主完成一个完整的数据大屏作品。

## 项目定位

- 适合前端初学者、数据可视化爱好者和学习者
- 以实战为主，帮助理解大屏布局、可视化组件和交互设计
- 后续会持续补充更多案例、组件和最佳实践

## 项目目标

- 通过一个可运行的项目，学习数据大屏的整体搭建思路
- 熟悉常见图表组件与可视化效果的实现方式
- 形成可持续维护和扩展的项目结构
